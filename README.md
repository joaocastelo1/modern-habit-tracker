## ✨ Key Features

- **📊 Dynamic Dashboard**: Real-time stats and completion rates.
- **📈 Weekly Progress**: Visual analytics using **Recharts** to track consistency.
- **📅 Consistency Calendar**: A visual representation of your daily streaks.
- **✨ Fluid UI/UX**: Smooth transitions and micro-animations powered by **Motion**.
- **🛠️ Full CRUD**: Create, read, update, and delete habits with a unified modal interface.
- **🌓 Adaptive Theme**: Sleek slate-based design with professional emerald accents.
- **📱 Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.

---

## 🛠️ Tech Stack

### Frontend
- **React 19**: Utilizing the latest hooks and concurrent features.
- **TypeScript**: Ensuring type safety and robust code maintainability.
- **Tailwind CSS 4**: Modern utility-first styling with high performance.
- **Lucide React**: Premium icon set for consistent visual language.
- **Motion (Framer)**: State-of-the-art animations and layout transitions.
- **Recharts**: Data visualization for habit performance analysis.

### Backend & Tooling
- **Node.js & Express**: Lightweight API handling.
- **PostgreSQL**: Production-ready data persistence (with in-memory fallback for demos).
- **Vite**: Ultra-fast build tool and development server.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/joaocastelo1/to-do-list-diario.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
